Bem-vindo ao TaskFlow

Gerencie suas tarefas, sprints e projetos com metodologias ágeis. Para começar, crie um novo projeto ou selecione um existente na barra lateral.

💡
Quadro Kanban — Arraste os cartões entre as colunas para atualizar o status das tarefas. Use "Nova Tarefa" para adicionar atividades e "Personalizar" para ajustar as colunas. Clique duas vezes em um cartão para ver detalhes, comentários e subtarefas.

💡
Backlog do Produto — Liste aqui todas as funcionalidades, melhorias e correções planejadas. Use "Adicionar Item" para incluir novas demandas. Itens refinados podem ser movidos para uma Sprint usando o botão 🏃.

Sprints — Ciclos de desenvolvimento com duração definida. Crie uma sprint com "Nova Sprint", mova itens do backlog para ela, depois use "Iniciar" para começar e "Finalizar" ao concluir.
